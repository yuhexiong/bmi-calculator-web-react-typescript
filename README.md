# BMI Calculator Web

A webpage where you can input your height and weight to calculate BMI.  

## Overview

- Language: TypeScript
- Web FrameWork: React


## Run

### install dependencies

```bash
npm install
```


### build

```bash
npm build
```

### run

```bash
npm start
```

## Pages

```bash
localhost:3000
```
